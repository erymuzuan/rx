# AssemblyAction

## Overview


With `Assembly` action you can execute your own custom code compiled into a .Net dll.

## Properties

| Property | Description |
| :------------- | :------------- |
| Name | The name fir your Assembly |
| Assembly | Select the assembly |
| Type | Choose a type from your assembly |
