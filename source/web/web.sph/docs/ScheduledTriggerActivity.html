ScheduledTriggerActivity

Overview

MUST BE SET as STARTUP ACTIVITY
Allows you to create a scheduled workflow. Scheduld task will be created for each enabled IntervalSchedule.
You can choose from 2 different schedules
* DailySchedule
* WeeklySchedule
* MonthlySchedule

Properties

Property Description
IntervalScheduleCollection - The list of schedules for this activity to run
