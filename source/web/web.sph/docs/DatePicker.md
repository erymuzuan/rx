#DatePicker
##Overview
DatePicker is a basic text input which allows your user to use a calendar like date picker to ease the datetime member type data entry. DatePicker will always store your date as text in sortable format i.e. `yyyy-MM-dd` eg `2012-02-08` without the time part.
