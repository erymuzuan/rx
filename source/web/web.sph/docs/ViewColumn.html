ViewColumn

Overview

Represent a column in a EntityView table, or ChildEntityListView

Properties















PropertyDescription
Path - The path to the member to bind the text to
Header - The displayed header for the column
Sort - Not Implemented
Format - The text format for your data, allows you to display in date , number or text format
IsLinkColumn - Enable the column to display a link to another EntityForm
IconCssClass - If set, an icon will appear before the link
