Button

Overview

Represents a button like form element, button allows you to perform an action

Properties

















PropertyDescription
Command - Is CommandName is present, this is the piece of Javascript code that will be executed when the button is clicked
UseClick - When unchecked, the Command scripts should return a Javascript promise object, to allow the code to run asynchronously. If checked the Command script runs sychnronously
CommandName -If no operation is selected, the command name will be the javascript function called when the button is clicked, "save" command will be automatically generated by Rx Developer
LoadingText - Reserved
CompleteText - Reserved
IconClass - Css Icon class for the font-awesome icons
IsToolbarItem - Makes the button appear in the toolbar area at the top of the screen, instead of at the dropped location
Operation - The operation name for an EntityDefinition, not applicable for Workflow screen . see EntityOperation for more details
