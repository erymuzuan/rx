EntityView

Overview

A set of user friendly tools to create a view for your entities. It allows you to display your entities in a table.

A tile will be displayed on the EntityDefinition landing page

You can also apply Filter to your list, which will naturally filter the list. List is read from ElasticSearch repository.

Sort can also be applied to your list.

ViewColumn allows you to specify the columns to your table. A leftmost column will automatically inserted with the value of your RecordName and a link point to the default form to your EntityDefinition


Drag to arrange columns, and click to select a column for you to edit the column's properties

Properties























PropertyDescription
FilterCollection - The list filter to apply to the list
ViewColumnCollection - The list of columns
SortCollection - Sorts your rows according to the criteria, pick a column and set the direction
ConditionalFormattingCollection
- Allows you to conditonally formatting your rows according to the Condition(Must be a valid Javascript expression e.g. Age > 10)
EntityViewId - Internal Id
IconClass - Not IMPLEMENTED
IconStoreId - The image icon for your tile dashboard
EntityDefinitionId - The id for the EntityDefinition
Name - Name
Route - Route to your view, must be unique
Note - Developers note
Query - Compiled query from your FilterCollections, this is normally generated by Rx Developer but you can always override it with your own Query DSL
IsPublished - make it available
Visibilty - (Reserved)
