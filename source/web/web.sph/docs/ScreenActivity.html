ScreenActivity

Overview

Properties

Property Description
Title - Name of the screen. Must be valid identifier
ViewVirtualPath -
WorkflowDefinitionId - The id of WorkflowDefinition which the screen belong to
InvitationMessageSubject - Message header to be sent to the performer of the sreen
InvitationMessageBody - Allow you to set message that will be sent by the system once the screen activity is executed
CancelMessageSubject - Message header to be sent to the performer of the sreen when the activity is cancel
CancelMessageBody - Allow you to set message that will be sent by the system when the screen activity is canceled
FormDesign - The design surface for the form - Use internally by `Rx Developer` web designer
Performer - Set access control for who can view the form
ConfirmationOptions - Allow you to set message to be prompt when the execution of the screen activity is done
IsPublic - Set the screen to be available for everyone
