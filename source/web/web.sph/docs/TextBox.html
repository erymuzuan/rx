TextBox

Overview

Represent a single line text input in your HTML. It's good for string, number or decimal.See other options for validation

Properties

Property Description
DefaultValue - Default value, is predefined value if no value is present in data
AutoCompletionEntity - The name of EntityDefinition from which you want the autocomplete to source from
AutoCompletionField - The field from the EntityDefinition from which the values will be taken
AutoCompletionQuery - The filter query to filter your EntityDefinitions
MinLength - Min character length for the text input
MaxLength - Max character length for the text input
